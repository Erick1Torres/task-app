# TaskFlow - Gestor de Tareas con Tailwind

Aplicaciones oara gestinar tareas prsonales, con soporte para **Modo Oscuro**.

## Demo en Vivo
https://task-app-two-lake.vercel.app/

---

## Características Principales

* **Diseño con Tailwind CSS:** Interfaz moderna utilizando un framework de utilidades de última generación.
* **Modo Oscuro (Dark Mode):** Alternancia de temas con persistencia en `localStorage`.
* **Persistencia de Datos:** Las tareas se guardan en el navegador, por lo que no se pierden al recargar.
* **Filtrado Dinámico:** Clasificación de tareas por categorías (Trabajo, Casa, Estudios) y búsqueda en tiempo real.
* **Totalmente Responsivo:** Adaptado para móviles, tablets y escritorio mediante clases de Tailwind.
* **Feedback Visual:** Transiciones suaves y efectos de hover en las tarjetas de tareas.

---

## Tecnologías Utilizadas

* **HTML5 Semántico:** Estructura clara con etiquetas `<header>`, `<main>`, `<aside>` y `<section>`.
* **Tailwind CSS:** Para el estilizado rápido, responsivo y el sistema de diseño.
* **JavaScript:** Lógica de manipulación del DOM, filtros y almacenamiento local.
* **Vercel:** Hosting y despliegue continuo.
* **Lucide icons:** biblioteca de iconos SVG de código abierto, gratuita y altamente personalizable.
